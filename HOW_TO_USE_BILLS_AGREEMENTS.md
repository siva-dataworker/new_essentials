# 📱 How to Use Bills & Agreements Feature

## Quick Start Guide for Accountants

---

## 🔐 Step 1: Login

1. Open the app
2. Login with accountant credentials
3. You'll see the site selection screen

---

## 📍 Step 2: Select Site

1. **Choose Area** from the dropdown
2. **Choose Street** from the dropdown (appears after area selection)
3. **Choose Site** from the dropdown (appears after street selection)
4. The app will automatically enter the site view

---

## 📄 Step 3: Access Bills & Agreements

Look at the **top-right corner** of the screen:
- You'll see a **receipt icon (📄)** next to the logout button
- **Tap the receipt icon** to open Bills & Agreements

---

## 📋 Step 4: Upload Documents

### The Bills & Agreements screen has 3 tabs:

### 📦 Tab 1: Material Bills
**For bills from material vendors (tiles shop, cement supplier, etc.)**

1. Tap the **"+ Add Bill/Agreement"** button (bottom-right)
2. Select **"Material Bill"**
3. Fill in the form:
   - Bill Number (required)
   - Bill Date (required)
   - Vendor Name (required)
   - Vendor Type (dropdown: Tiles Shop, Cement Supplier, etc.)
   - Material Type (dropdown: Tiles, Cement, Steel, etc.)
   - Quantity (required)
   - Unit (dropdown: sqft, bags, kg, etc.)
   - Unit Price (required)
   - Tax Amount (optional)
   - Discount Amount (optional)
   - Payment Status (PENDING, PARTIAL, PAID)
4. **Select PDF File** (tap the button to choose file)
5. Tap **"Upload"**
6. Wait for success message ✅

**The app will automatically calculate:**
- Total Amount = Quantity × Unit Price
- Final Amount = Total Amount + Tax - Discount

---

### 🏗️ Tab 2: Vendor Bills
**For bills from service providers (contractors, electricians, etc.)**

1. Tap the **"+ Add Bill/Agreement"** button
2. Select **"Vendor Bill"**
3. Fill in the form:
   - Bill Number (required)
   - Bill Date (required)
   - Vendor Name (required)
   - Vendor Type (dropdown: Contractor, Electrician, Plumber, etc.)
   - Service Type (required, e.g., "Electrical Wiring")
   - Service Description (optional)
   - Amount (required)
   - Tax Amount (optional)
   - Discount Amount (optional)
   - Payment Status (PENDING, PARTIAL, PAID)
4. **Select PDF File**
5. Tap **"Upload"**
6. Wait for success message ✅

**The app will automatically calculate:**
- Final Amount = Amount + Tax - Discount

---

### 📝 Tab 3: Site Agreements
**For signed agreements (customer agreements, contractor agreements, etc.)**

1. Tap the **"+ Add Bill/Agreement"** button
2. Select **"Site Agreement"**
3. Fill in the form:
   - Agreement Type (dropdown: Site Agreement, Contractor Agreement, etc.)
   - Agreement Number (optional)
   - Agreement Date (required)
   - Party Name (required, e.g., customer name)
   - Party Type (dropdown: Customer, Contractor, Vendor, etc.)
   - Agreement Title (required)
   - Description (optional)
   - Contract Value (optional)
   - Start Date (optional)
   - End Date (optional)
4. **Select PDF File**
5. Tap **"Upload"**
6. Wait for success message ✅

---

## 👀 Step 5: View Uploaded Documents

### Each tab shows:
- **Card-based list** of all documents
- **Color-coded status badges**:
  - 🔴 PENDING (red)
  - 🟠 PARTIAL (orange)
  - 🟢 PAID (green)
- **Vendor/Party information**
- **Bill numbers and dates**
- **Amounts**

### To view a PDF:
1. **Tap on any card** in the list
2. The PDF will open in your device's PDF viewer

### To refresh the list:
1. **Pull down** on the list to refresh
2. New documents will appear

## 💡 Tips

### ✅ Do's:
- Always select the correct vendor/material type
- Enter accurate bill numbers for tracking
- Upload clear, readable PDF files
- Update payment status when payments are made
- Add notes for important information

### ❌ Don'ts:
- Don't upload non-PDF files (only PDF supported)
- Don't skip required fields (marked with *)
- Don't forget to select the PDF file before uploading
- Don't close the dialog while uploading

---

## 🔍 Troubleshooting

### Problem: "Please select a PDF file" error
**Solution:** Tap the "Select PDF File" button and choose a PDF from your device

### Problem: "Please fill all required fields" error
**Solution:** Check that all fields marked with * are filled in

### Problem: PDF doesn't open
**Solution:** 
1. Make sure you have a PDF viewer app installed
2. Check your internet connection
3. Try refreshing the list and tapping again

### Problem: Upload fails
**Solution:**
1. Check your internet connection
2. Verify the PDF file is not corrupted
3. Try uploading again
4. Contact support if issue persists
