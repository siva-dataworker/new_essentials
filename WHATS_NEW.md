## 🚀 How to Use

### Manage Users:
1. Open Profile
2. Click "Manage Users"
3. See pending requests
4. Click Approve/Reject
5. Switch to "All Users" tab

### Create Site:
1. Open Profile
2. Click "Create Site"
3. Enter Area, Street, Site
4. Click Create
5. Site appears in dropdown
